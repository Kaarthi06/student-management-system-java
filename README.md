# Student Management System

A Java console-based Student Management System built using Core Java.

## Features

- Add Student
- Update Student
- Delete Student
- Search Student
- Display Students
- Sort Students
- File Handling Persistence
- User Registration & Login
- SHA-256 Password Hashing
- Salt-Based Password Security
- Password Strength Validation
- Account Blocking After 3 Failed Attempts
- ACTIVE/BLOCKED User Status Management

## Technologies Used

- Java
- OOP
- ArrayList
- File Handling
- SHA-256 Hashing
- UUID Salt Generation

## Project Structure

src/com/kaarthi/student

- Main.java
- Student.java
- StudentService.java
- FileService.java
- AuthService.java
- User.java
- ValidationService.java

## How to Run

1. Clone the repository
2. Open in IntelliJ IDEA
3. Run Main.java
